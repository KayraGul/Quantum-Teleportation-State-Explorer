# Quantum Teleportation State Explorer

A computational quantum optics project for studying continuous-variable quantum teleportation, Wigner-function diagnostics, noisy Gaussian teleportation channels, and constrained optimization of quantum states in a truncated Fock basis.

## Overview

This project studies noisy continuous-variable quantum teleportation using phase-space methods and Fock-basis state optimization. The work includes Wigner-function diagnostics, teleportation fidelity calculations, Wigner negativity analysis, comparison with reference states, and robustness tests under realistic noise models.
